# TreximAI — Фінансова Модель

Інтерактивна фінансова модель TreximAI в одному HTML-файлі.

## Функції

- **Параметри моделі** — змінюєш поля, таблиця і графіки перераховуються автоматично
- **Break-even аналіз** — два сценарії (оптимістичний / песимістичний)
- **Помісячний прогноз** — виручка, витрати, cash flow, накопичений CF
- **Графік Chart.js** — revenue / burn / cum. CF / кількість компаній
- **💾 Зберегти версію** — зберігає поточні значення параметрів у `localStorage`
- **📂 Версії** — перегляд, завантаження або видалення збережених версій
- **🖨 Друк / PDF** — оптимізовані стилі друку (A4), автоматично вставляє блок параметрів

## Запуск

Відкрий `index.html` у браузері. Без build-крока, без залежностей крім CDN Chart.js.

## Зберігання версій

Версії зберігаються у `localStorage` браузера під ключем `trexim_fin_versions_v1`.
Дані прив'язані до браузера/профілю/домену — очищення кешу їх видалить.

## Друк у PDF

`Ctrl+P` → "Зберегти як PDF". Параметри моделі автоматично виводяться зверху сторінки.
